### 2. Примеры SQL-запросов
В ходе учебных и практических задач регулярно применяю SQL для верификации данных на бэкенде.

```sql
-- Поиск пользователя в базе данных по его email
SELECT * 
FROM users 
WHERE email = 'test@example.com';

-- Вывод имён пользователей и дат их завершённых заказов (объединение таблиц)
SELECT 
    u.name,
    o.order_date
FROM users u
JOIN orders o ON u.id = o.user_id
WHERE o.status = 'completed';
```

### 3. Пример чек-листа (Авторизация)
Базовый пример высокоуровневой структуры проверок для модуля аутентификации:
* [ ] Проверить авторизацию с валидными учётными данными (Happy Path).
* [ ] Проверить вывод ошибки при вводе неверного пароля.
* [ ] Проверить логику восстановления доступа через форму "Забыли пароль".
